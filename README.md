## Prerequisites

Before diving into operating system development in general, the learner should be competent in the theory and practice of the following :

- Programming in C and Assembly x86
- Architecture of x86 computers
- Boot process of a computer

Personal recommendations for learning the previous topics :

- Hardware : [BranchEducation](https://www.youtube.com/@BranchEducation/videos)
- Hardware, Operating System and Low Level :
[CoreDumped](https://www.youtube.com/@CoreDumpped), [GeeksForGeeks](https://www.geeksforgeeks.org/operating-systems/operating-systems/), [GeeksForGeeks](https://www.geeksforgeeks.org/computer-science-fundamentals/computer-hardware/)
- Assembly : [TryHackMe](https://tryhackme.com/)
